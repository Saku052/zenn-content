# Rustにおけるマクロって何なんだろう
簡単に言うと、Rustの長くなりがちなコードを簡単に1行で書けるようになる機能の事。
例えば、

``` rust 
println!("Hello, world!");
```

``` rust
{
    ::std::io::_print(::core::fmt::Arguments::new_v1(
        &["Hello, world!\n"],
        &[],
    ));
}
```

上記二つのコードは挙動で言えば全く同じ事を行います。
しかし、第一のコードの方が短くて読みやすいですね。
マクロはこのように、冗長なコードを簡潔に書くのに役立ちます。

仕組み的な部分でいうと、マクロはコンパイル時に展開されます。つまり、内部的にマクロを使用しない方が優れていると言う事
もなく、本質的にはどちらも同じコードが生成されます。

# おまけ

### cargo expandについて

おまけですが、マクロの動作を確認するのに便利なツールがあります。
それが `cargo expand` です。このコマンドは指定したファイルに含まれるマクロを展開してくれるます。デバック時とかで役に立つみたいです。
